# SOC Incident Response Playbook (Beginner)

## Overview
This project demonstrates beginner-level incident response workflows based on
CISA Cyber Defense Analyst training and NIST Incident Response guidelines.

## Scope
- Alert triage
- Incident classification
- Containment strategies
- Escalation criteria
- Lessons learned

## Incident Types Covered
- Phishing
- Brute-force authentication attacks
- Malware infections

## Frameworks Referenced
- NIST SP 800-61
- MITRE ATT&CK
- CISA Incident Response guidance

## Disclaimer
This project is for educational purposes only. No real production data is used.
