# Phishing Incident Response

## Detection
- SIEM alert for suspicious email activity
- User-reported phishing email

## Triage
- Verify sender domain
- Inspect email headers
- Check links and attachments

## Containment
- Block sender domain
- Quarantine affected mailbox
- Reset user credentials if needed

## Eradication
- Remove malicious emails from mailboxes
- Update email filtering rules

## Recovery
- Monitor account activity
- Educate user on phishing indicators
